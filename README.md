## Wrapper for PyGithub package

This repo is a sample repo containing some examples about how to sign commits automatically using GitHub.

## Build and run

Build docker image:
```
./bin/build.sh
```

Run all examples in `src/main.py`:
```
./bin/run-main.sh
```

Run a given example:
```
./bin/run-01.sh
./bin/run-XX.sh
```
